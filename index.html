Georgia Tech Registration Alerts

The only thing more competitive than classes at Georgia Tech is registering for those classes. I wrote this command-line program to automatically check the registration webpage and send me text message alerts when a seat in a class I need has opened up.

  • Languages: Java, MATLAB (network latency visualization), jsoup and google-voice-java libraries

Motion-Controlled RC Helicopter

This $20 RC Helicopter is pretty cool, but the RC controller it shipped with is hard to use. Using an arduino programmed with C and AVR assembly and an IR LED, I created a new controller that uses tilt controls to fly the helicopter.

  • Languages: C, AVR Assembly, MATLAB/Simulink (testing/debugging)
